# TopstepAlgo-C-Trading-Framework-for-TopstepX
TopstepAlgo est un framework C# pour créer des algorithmes de trading sur TopstepX sans manipuler directement l’API. Il gère connexion, contrats, ordres, positions et données marché afin que le développeur se concentre uniquement sur la stratégie (scalping, breakout, hedge, pair trading).
